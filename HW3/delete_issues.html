Deleting an issue is an action that only the issue initiator and site admin can utilize. It should be used when the issue is considered irrelvant, misleading, posted by accident or by any way violates the site's Terms of Service. If you see an issue that violates our Terms of Service please report it to the site admins.

Issues that are not in consideration or are duplicates should be closed. Issues that are fixed, implemented or otherwise solved should be resolved.

Please note that deleting an issue is a permanent action; an issue and any communication history related to it would not be recoverable once deleted. Closing/resolving on the otherhand would merely change the status of the issue, so that it can be reopened or disputed and that its history may be viewed.
